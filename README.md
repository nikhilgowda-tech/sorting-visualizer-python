# Sorting Algorithm Visualizer

A Python project that visualizes sorting algorithms using Matplotlib.

## Algorithms Included
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort

## Technologies Used
- Python
- Matplotlib
- Google Colab

## Features
- Real-time sorting animation
- Color visualization
- Execution time calculation
